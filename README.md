# 🎓 Student Management System (React)

A React-based Student Management System with student list, favourites system, registration form, contact form, login and signup functionality using Context API and React Router.

---

## 🚀 Features

### 🔐 Authentication System
- User Signup
- User Login
- Context API used for user state management
- Navigation using React Router

---

### 🎓 Student Management
- Display student list from static data
- Add students to favourites
- Remove students from favourites
- Heart icon toggle (liked / unliked state)
- Button disables after adding to favourites

---

### 📝 Registration Form
- Controlled input fields
- Required field validation
- Success alert after submission
- Button disables after successful registration

---

### 📞 Contact Form
- Email input validation
- Subscribe functionality
- Button disables after successful subscription

---

### 🧭 Routing System
- React Router navigation between pages:
  Signup → Login → Home → Student List → Favourite Students

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- Context API
- JavaScript (ES6+)
- Tailwind CSS / CSS

---

## 🎯 How it works

- User signs up or logs in
- After login, user is redirected to Home page
- Student list is loaded from static data
- Users can add/remove favourites
- Favourite list updates instantly using Context API
- Forms validate input before submission
- Buttons disable after successful actions

---

## 🌟 Future Improvements

- Backend integration (Firebase / Node.js)
- Persistent login using localStorage
- Protected routes (auth guard)
